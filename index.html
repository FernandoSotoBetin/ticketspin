Acerca De TicketSpin v1.0.0

TicketSpin es un generador de ticket que consta de 3 opciones básicas aleatorias que son: Lista De Números, El Abecedario y Lista Editable.

Opción Número: Al seleccionar esta opción se activa una entrada para digitar el numero máximo a generar.

Opción Abecedario: Genera una de las 27 letras del Abecedario.
Opción Lista: Está es la opción más interesante ya que se pueden resumir las dos opciones anteriores, por lo que se puede ingresar, en el area de texto los datos a generar, cada linea corresponde a un registro así Por Ejemplo: Son 3 registros independientes, que se deben registrar con el botón Registrar
Finalmente tenemos los botones de Generar para obtener los datos aleatorios deseados, Limpiar para resetear los valores por defecto y Limpiar Lista Elimina todos los registros de la lista Editable.
